# DP-600 Study Guide

Visual, study guide for the **Microsoft DP-600 Fabric Analytics Engineer** certification exam.

**Live site → [livof.github.io/DP-600](https://livof.github.io/DP-600/)**

## What is this?

Short pages. Big visuals. No walls of text. Each topic covers one exam concept with:

- A visual flow diagram
- Plain-language explanation with analogies
- The actual exam question (tap to answer)
- Why every wrong answer fails
- Trigger words for pattern recognition
- Practice quiz with score tracking

Built one topic at a time as I study — using [Claude](https://claude.ai) as a learning companion.

## Exam domains

1. **Plan, implement, and manage a solution** (25–30%)
2. **Prepare and serve data** (25–30%)
3. **Implement and manage semantic models** (25–30%)
4. **Explore and analyze data** (10–15%)
5. **Monitor and optimize** (5–10%)

## How to use

Open the [live site](https://livof.github.io/DP-600/) and tap a topic. Each page is self-contained — no frameworks, no build step, just HTML.

## How it's built

Each page is a single `.html` file with inline CSS and JS. No dependencies. Designed for GitHub Pages — just push and it works.
