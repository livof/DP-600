Exam Tips

🔀
Direct Lake vs Import
Direct Lake reads Delta files directly from OneLake — no import needed. Switches to DirectQuery when row-level security is active.
🔗
Shortcuts vs Pipelines
Shortcuts reference data in place — zero copy. Pipelines physically move the data. Key distinction for the exam.
📦
Git Integration
Only Azure Repos (same tenant) was supported originally. GitHub support was added in September 2024 — watch out for version of this question.
⚡
Capacity SKUs
F-SKUs (F2–F2048) are pay-as-you-go. Research division = on-demand capacity with per-minute billing. Fabric Trial = F64 equivalent.
🏗️
Least Privilege
A recurring requirement in case studies. Always pick the role that grants minimum needed access — Viewer, not Member; Reader, not Admin.
🧊
Delta Lake Default
Delta is the default and preferred table format in Microsoft Fabric. Managed tables live inside the lakehouse. External tables point outward.
