High-yield traps
Direct Lake
Falls back to DirectQuery silently
When row-level security is active on a warehouse, Direct Lake mode automatically falls back to DirectQuery — without warning. Expect questions on when this happens.
Shortcut vs Pipeline
Zero copy vs physical move
Shortcuts reference data in place — nothing is copied. Pipelines physically move data. "Minimize duplication" = shortcut. "Load into lakehouse" = pipeline.
Git Integration
Azure Repos originally; GitHub added Sep 2024
Questions created before Sept 2024 have C (Azure Repos) as the only correct answer. Newer questions may accept both. Read the question date if shown.
Least Privilege
Always appears in case studies
Pick the minimum role needed. Viewer over Member. Contributor over Admin. SQL endpoint read = specific workspace role, not admin access.
Managed vs External
Managed tables live inside the lakehouse
Managed tables: Fabric controls storage. Dropping the table deletes data. External: you control storage. Dropping the table leaves the files. Delta is always the format.
Capacity SKUs
On-demand = Fabric Trial or F-SKU
Research division "per-minute billing" = Fabric Trial capacity equivalent (F64). F2–F2048 are pay-as-you-go. P-SKUs (Power BI Premium) don't support all Fabric workloads.
