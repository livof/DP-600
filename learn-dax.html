Concepts - read these first

🧮
CALCULATE changes the filter context around a measure

CALCULATE([Measure], Filter) evaluates a measure inside a modified filter context. Time intelligence functions like SAMEPERIODLASTYEAR only produce a filter - they always need to be wrapped in CALCULATE to actually compute something.

📁
PBIT is a template with no data. PBIP is a project for Git.

PBIX is the standard file and contains data. PBIT is a template with structure and connections but no data - used to share report layouts. PBIP is a 📁-based project for version control. PBIDS is just a data source connection file with no report.

💱
Dynamic format strings change the 💱 symbol based on context

A dynamic format string is a DAX expression that returns a format string. Use it when a measure should display in different currencies based on a user selection. Field parameters switch which measure shows - not how it is formatted.

🔍
Model view shows column properties. Model Explorer shows the full hierarchy.

Model view (the diagram tab) is best for checking column properties like Summarize By - spot surrogate keys set to Sum by mistake. Model Explorer (the tree) shows measures and 🧮ulation groups. Performance Analyzer records query timings per visual.

🚀
NOT ISEMPTY() is faster than COUNTROWS() > 0

ISEMPTY short-circuits as soon as it finds one row. COUNTROWS scans the whole table first, then compares. For existence checks in DAX, always use ISEMPTY - it is measurably faster on large tables.
